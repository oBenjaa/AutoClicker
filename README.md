# AutoClicker criado em Python 

## Como utilizar:
- **Ter o [Python](https://www.python.org/downloads/) instalado em sua máquina**
- **Abra uma nova pasta em um editor de código de sua preferência**
- **Clone o repositório usando o seguinte comando:**
- ````git clone git@github.com:oBenjaa/AutoClicker.git````
-  **Instale as biblioteca Pynput usando os seguinte comando:**
"pip install pypunt"
